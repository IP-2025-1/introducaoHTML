Introdução ao HTML

Tags

Todo o arquivo HTML é composto por tags, as tags podem ser de dois tipo, abertura e fechamento ou de fechamento automático

Abertura e fechamento: para declarar uma tag de abertura e fechamento, deve-se fazer o seguinte: <tag> conteúdo da tag</tag>

Fechamendo automático: para declarar uma tag de fechamento automa1tico, deve-se fazer o seguinte: <tag/>

Elementos comuns no HTML

Títulos

O HTML possui 6 títulos, do h1 até o h6. Sendo h1 o maior e h6 o menor

Para se declarar um título x, basta usar a tag <hx>Título</hx>

Parágrafo

Para se declara um parágrafo no HTML, deve-se utilizar a tag <p>Conteúdo do parágrafo</p>

Divs

Uma div é um contâiner para outros elementos

Para se declara uma div no HTML, deve-se utilizar a tag <div>Conteúdo da div</div>

Por exemplo:

        <div>
        <h1>Lorem ipsum</h1>
        <p>Lorem ipsum dolor sit, amet consectetur adipisicing elit. Eum eius nobis placeat minima sint sapiente saepe quas
        magnam est, asperiores in magni debitis excepturi voluptate ipsum sunt unde hic deleniti?</p>
        <button>Lorem</button>
        </div>

Botões

Para se declarar um botão, basta usar a tag <button>Texto do botão</button>

Botões tem alguns atributos disponíveis, os princiapis são "onClick" e "type"

O atributo onClick define a função que será executada assim que o botão for pressinado Esta função geralmente é escrita em JavaScript

O atributo type define o tipo do botão, podendo ser "button", "menu", "submit" ou "reset"

Links

Para se declarar um link, basta usar a tag <a href="url-do-link">Texto do link</a>

Obs.: Diferentemete dos atributos "onClick" e "type" dos botões que são opcionais, o atributo href numa tag âncora é essencial para o funcionamento
Google

A tag a tem um atributo opcional chamado "target", quando este recebe "_blank", o link é aberto em uma nova página do navegador

Formulários

Formulários são como caixas organizadoras para inputs

Obs.: Num formulário, caso haja botões sem o atributo type definido, eles são tratados como botões de type="submit" e enviam automaticamente o formulário em caso de clique

O formulário tem um atributo opcional chamado action, ele diz para onde os dados serão enviados.

Para se declarar um formulário, basta usar a tag <form>Conteúdo do formulário</form>

Por exemplo:

        <form>
        <input type="text" placeholder="Digite seu nome aqui"/>
        <button>Enviar</button>
        </form>
