Introdução ao HTML

Tags

Todo o arquivo HTML é composto por tags, as tags podem ser de dois tipo, abertura e fechamento ou de fechamento automático

Abertura e fechamento: para declarar uma tag de abertura e fechamento, deve-se fazer o seguinte: <tag> conteúdo da tag</tag>

Fechamendo automático: para declarar uma tag de fechamento automa1tico, deve-se fazer o seguinte: <tag/>

Elementos comuns no HTML

Títulos

O HTML possui 6 títulos, do h1 até o h6. Sendo h1 o maior e h6 o menor

Para se declarar um título x, basta usar a tag <hx>Título</hx>

Parágrafo

Para se declara um parágrafo no HTML, deve-se utilizar a tag <p>Conteúdo do parágrafo</p>

Divs

Uma div é um contâiner para outros elementos
